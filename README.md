Data Cleaning and Preprocessing

In this task, I worked on cleaning and preparing a dataset for machine learning. I used the Titanic dataset for this work.

First, I loaded the dataset using pandas and checked how the data looks. I displayed the first few rows to understand the structure.

Then I checked for missing values in the dataset. Wherever values were missing, I filled them using the median so that the data becomes complete.

After that, I handled categorical data. Some columns had text values, so I converted them into numbers using label encoding.

Next, I applied feature scaling using StandardScaler. This step helps in bringing all values to a similar range, which is useful for machine learning models.

I also printed the dataset before and after preprocessing to clearly see the changes.

Tools used:
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

Finally, the dataset is cleaned and ready to be used for machine learning.

Through this task, I understood how important preprocessing is and how it helps in improving model performance.
